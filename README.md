## About The Project

A XOX game made using React and Tailwind


### Installation and Usage

1. Clone the repo
   ```sh
   git clone https://github.com/ozanmrt/tic-tac-toe.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start
   ```js
   npm start
   ```
