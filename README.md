## Installation and Usage

1. Clone the repo
   ```sh
   git clone https://github.com/ozanmrt/tic-tac-toe.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start
   ```js
   npm start
   ```
